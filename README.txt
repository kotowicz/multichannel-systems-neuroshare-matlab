 Multichannel Systems file loading routines for Matlab
=======================================================

- works on Windows, MacOS and Linux.
- based on http://www.multichannelsystems.com/software/neuroshare version 3.4.
- use 'load_multichannel_systems_mcd.m' and 'load_multichannel_systems_raw.m' for convenient data access.
- compile instructions for Windows and Linux are found in 3rdparty/MatLab/Matlab-Import-Filter/Mex_Sources/ directory.
- see also 3rdparty/README.txt file.
